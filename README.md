** Modern Dark Calculator**
A sleek, responsive, and functional web-based calculator built with HTML5, CSS3, and Vanilla JavaScript. This project features a modern glassmorphism-inspired UI with a dark theme and smooth interactions.

** Live Demo**
Check out the live project here: https://sameer009988.github.io/calcultor/

** Features**
Basic Arithmetic: Addition, Subtraction, Multiplication, Division, and Percentage.

Real-time Display: Large, easy-to-read input display with right-aligned text.

Advanced Controls:

AC: Clear the entire input.

DEL: Backspace functionality to remove the last digit.

=: Instant calculation using JavaScript logic.

Modern Design: * Dark gradient background.

Circular buttons with hover effects and soft shadows.

Clean typography using the "Roboto" Google Font.

 Technologies Used
HTML5: Structure and layout.

CSS3: Styling, Flexbox for centering, and CSS gradients.

JavaScript (ES6): Event listeners, DOM manipulation, and calculation logic using eval().

Project Structure
├── index.html   # Main HTML structure
├── style.css    # UI styling and layout
└── script.js    # Logic for calculations and button clicks
